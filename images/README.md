# Images Folder

Place your 360° rotation images in this folder.

## Naming Convention

Your images should be named sequentially:
- `chair_001.jpg`
- `chair_002.jpg`
- `chair_003.jpg`
- ... (up to `chair_036.jpg` for 36 images)

## Image Requirements

1. **Consistent Naming**: All images must follow the same naming pattern
2. **Sequential Numbers**: Frame numbers should be sequential (001, 002, 003, etc.)
3. **Same Format**: All images should use the same file format (jpg, png, or webp)
4. **Even Spacing**: Images should be evenly spaced around the product:
   - 36 images = 10° per image
   - 72 images = 5° per image
5. **Consistent Settings**: Same camera distance, lighting, and exposure for all images
6. **Same Aspect Ratio**: All images should have the same dimensions/aspect ratio

## Example Structure

```
images/
  ├── chair_001.jpg
  ├── chair_002.jpg
  ├── chair_003.jpg
  ├── ...
  └── chair_036.jpg
```

## Tips

- Use high-quality images (at least 1200px width recommended)
- Optimize images for web to reduce load time
- Keep file sizes reasonable (under 500KB per image if possible)
- Use consistent backgrounds or transparent backgrounds
